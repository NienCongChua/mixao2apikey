Config Overview Chatbot
Endpoint: http://localhost:8080/v1/chat/completions

Chat UI: http://localhost:8080/chat · Overview: http://localhost:8080/overview

📖 Cách lấy request đăng nhập

1. Vào aistudio.xiaomimimo.com và đăng nhập
2. Mở DevTools (F12) → Tab Network
3. Gửi 1 tin nhắn bất kỳ, tìm request /open-apis/bot/chat
4. Right-click → Copy as cURL
5. Dán vào ô bên dưới và bấm Parse cURL

🔑 API keys sử dụng endpoint (optional)

Khi có ít nhất một API key active, các endpoint /v1/* sẽ yêu cầu header Authorization: Bearer <key>. Nếu không tạo key và không set MIMO_API_KEY, server vẫn chạy không auth như trước.

Chưa có API key nào.

🚀 Sử dụng

Ví dụ với Python OpenAI SDK:

from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:8080/v1",
    api_key="sk-your-key"  # Bỏ qua nếu chưa bật auth
)

response = client.chat.completions.create(
    model="mimo-v2.5-pro",
    messages=[{"role": "user", "content": "Hello!"}],
    stream=True
)

for chunk in response:
    print(chunk.choices[0].delta.content or "", end="")
